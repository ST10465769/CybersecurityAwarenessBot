🛡️ Cybersecurity Awareness Bot

📌 Project Overview
The Cybersecurity Awareness Bot is a console-based application designed to educate users about cybersecurity best practices.

🎯 Features
- User name input and personalised greeting
- Voice greeting
- Chatbot responses
- ASCII art interface
- Input validation
- Improved console UI

🛠️ Technologies Used
- C#
- .NET
- Visual Studio Code

 ▶️ How to Run
1. Clone the repo
2. Open in VS Code
3. Run:
   dotnet run
